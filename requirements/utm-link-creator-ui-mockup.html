UTM 링크 생성 UI 개선 목업

목표: 운영자가 “무엇을 왜 입력하는지” 바로 이해하고, 생성 전 추적 품질을 확인할 수 있는 폼.

1. 필드 구분 강화 필수, 권장, 광고 바인딩을 카드와 좌측 색상으로 분리합니다.
2. 필드마다 의미 제공 각 입력에 예시, 성과 연결 위치, 상태 뱃지를 함께 둡니다.
3. 실시간 URL 프리뷰 입력 즉시 어떤 파라미터가 생기는지 오른쪽에서 확인합니다.
4. 품질 점수 필수만 채운 링크와 분석 가능한 링크를 운영자가 구분합니다.

새 UTM 링크 생성

빠른 입력은 유지하되, 분석에 필요한 필드를 놓치지 않게 구성합니다.

82
분석 가능성 높음

필수값, UTM ID, 플랫폼, 소재 형식이 채워졌습니다. 광고 ID가 있으면 100점에 가까워집니다.

필수 입력

이 값이 없으면 링크가 만들어지지 않습니다. 프리셋 선택 시 source와 medium은 자동 입력됩니다.

필수 4개
채널 source + medium 자동
왜 필요? 유입이 시작된 장소입니다. 예시 instagram, meta, everytime, poster
캠페인 utm_campaign
왜 필요? 예산과 성과를 묶는 기준입니다. 권장 형식 YYYYMM_objective_audience
소스 utm_source
성과 연결 채널별 유입, 가입, 결제 리포트의 1차 그룹입니다.
매체 utm_medium
성과 연결 paid, organic, social, offline 같은 트래픽 유형 비교에 씁니다.
웹 랜딩 전체 UTM query가 URL에 붙습니다. 가장 직관적인 프리뷰를 제공합니다.
iOS App Store App Store의 ct 캠페인 토큰 중심으로 이동합니다. 추가 추적은 리다이렉트 이벤트에서 보강합니다.
Android Play Store referrer에 전체 UTM 파라미터를 인코딩해서 전달합니다.

권장 입력

필수는 아니지만, 소재별/타겟별/전술별 성과를 보려면 사실상 필요합니다.

성과 분석 강화
콘텐츠 utm_content
왜 필요? 같은 캠페인 안에서 소재 A/B를 구분합니다.
타겟/키워드 utm_term
예시 school_a, search_keyword, female_univ_20s
UTM ID 자동 생성
왜 필요? 플랫폼이 달라도 같은 캠페인을 안정적으로 묶습니다.
Source platform 플랫폼
주의 source=instagram이어도 실제 구매 매체는 Meta Ads일 수 있습니다.
Creative format 소재 형식
성과 연결 이미지, 영상, QR 등 형식별 전환율 비교에 씁니다.
Marketing tactic 전술
왜 필요? 운영 언어와 대시보드 필터 기준을 맞춥니다.

광고 플랫폼 바인딩

Meta/Google 관리자 화면의 ID를 연결하면 광고 단위 drilldown과 전환 내보내기가 쉬워집니다.

고급 추적
플랫폼 binding
UI 제안 플랫폼에 따라 Adset 또는 Ad group 필드만 보여줍니다.
Campaign ID 권장
복붙 지원 광고 URL이나 텍스트에서 자동 추출 후보를 보여줄 수 있습니다.
Ad ID / Creative ID 선택
언제? 같은 캠페인 안에 소재가 많을 때 개별 광고 성과를 분리합니다.

구현 시에는 기존 `TextField`/`Chip`/`Alert` 기반으로 옮기되, 섹션 컨테이너와 오른쪽 프리뷰 패널을 추가하는 방향이 가장 작습니다.

실시간 URL 프리뷰

https://some-in-univ.com? utm_source=instagram &utm_medium=cpc &utm_campaign=2026_may_signup_kr20f &utm_content=reels_video_a &utm_term=female_univ_20s &utm_id=instagram_2026_may_signup_kr20f
utm_source채널 성과
utm_campaign캠페인 성과
utm_content소재 A/B
utm_id플랫폼 통합 키

생성 전 검증

좋음
캠페인명에 기간, 목적, 타겟이 포함되어 있습니다.
좋음
source=instagram, sourcePlatform=Meta Ads 조합은 유효합니다.
보완 가능
Campaign ID를 넣으면 광고 관리자 성과와 더 안정적으로 연결됩니다.
