# 어드민 대시보드 기능 릴리즈 노트

> **기간**: 2025년 12월 25일 ~ 2026년 1월 3일  
> **대상**: PM, 마케터, 운영팀

---

## 릴리즈 개요

이번 업데이트에서는 **매출 분석 강화**, **iOS 환불 관리**, **회원 심사 개선**, **리텐션 관리 도구** 등 운영 효율성을 높이는 다양한 기능이 추가되었습니다.

---

## 신규 기능

### 1. iOS 환불 관리

| 항목 | 내용 |
|------|------|
| **해결하는 문제** | Apple App Store 인앱 결제 환불 현황을 별도로 확인할 수 없었음 |
| **할 수 있는 것** | iOS 환불 내역 조회, 환불 상태별 필터링, 기간별 검색 |
| **사용 대상** | 운영팀, CS팀 |
| **메뉴 위치** | 관리자 > iOS 환불 관리 |

**주요 기능**
- 환불 상태 (정상/환불완료) 필터링
- 사용자명, 거래ID로 검색
- 날짜 범위 지정 조회
- 환불 상태 동기화 버튼

---

### 2. 결제 성공률 분석

| 항목 | 내용 |
|------|------|
| **해결하는 문제** | 결제 시도 대비 성공률을 파악하기 어려웠음 |
| **할 수 있는 것** | 결제 성공률 시각화, 실패 원인 분석 |
| **사용 대상** | PM, 마케터 |
| **메뉴 위치** | 관리자 > 매출 조회 |

---

### 3. 파묘 좋아요 관리 (리텐션)

| 항목 | 내용 |
|------|------|
| **해결하는 문제** | 장기 미접속 여성 유저에게 온 좋아요가 방치되어 매칭 기회 손실 |
| **할 수 있는 것** | 미접속 유저 대신 좋아요 확인/거절 처리, 처리 이력 조회 |
| **사용 대상** | 운영팀 |
| **메뉴 위치** | 관리자 > [리텐션] 파묘 좋아요 |

**주요 기능**
- 7일 이상 미접속 + 구슬 보유 여성 유저 목록
- 미확인 좋아요 수 확인
- 대리 조회/거절 처리 (거절 비율 조정 가능)
- 쿨다운 시스템 (과도한 처리 방지)
- 처리 이력 로그 조회

---

### 4. 탈퇴 여성 회원 복구 (리텐션)

| 항목 | 내용 |
|------|------|
| **해결하는 문제** | 탈퇴한 여성 회원 재유입 수단 부재 |
| **할 수 있는 것** | 탈퇴 여성 회원 목록 조회, 계정 복구 처리 |
| **사용 대상** | 운영팀 |
| **메뉴 위치** | 관리자 > [리텐션] 탈퇴 회원 복구 |

---

### 5. 배너 관리

| 항목 | 내용 |
|------|------|
| **해결하는 문제** | 앱 내 배너 교체 시 개발팀 요청 필요 |
| **할 수 있는 것** | 홈/모먼트 배너 직접 등록, 수정, 순서 변경, 활성화/비활성화 |
| **사용 대상** | 마케터, 운영팀 |
| **메뉴 위치** | 관리자 > 배너 관리 |

**주요 기능**
- 이미지 업로드 및 미리보기
- 배너 클릭 시 이동할 URL 설정
- 노출 기간 설정 (시작일/종료일)
- 드래그앤드롭 순서 변경

---

### 6. 국가 선택 기능

| 항목 | 내용 |
|------|------|
| **해결하는 문제** | 한국/일본 데이터를 구분해서 볼 수 없었음 |
| **할 수 있는 것** | 한국(KR) / 일본(JP) 데이터 전환 |
| **사용 대상** | 전체 어드민 사용자 |
| **메뉴 위치** | 화면 우측 하단 플로팅 버튼 |

## 개선된 기능

### 회원 적격 심사

| 개선 항목 | 이전 | 이후 |
|----------|------|------|
| 심사 참고 정보 | 없음 | 매칭 히스토리, 신고 이력 등 표시 |
| 거절 이미지 | 텍스트만 | 거절 사유 이미지 미리보기 |
| 성별 뱃지 | 간헐적 오류 | 정상 표시 |

---

### 카드뉴스 관리

| 개선 항목 | 이전 | 이후 |
|----------|------|------|
| 미리보기 | 저장 후 확인 | 실시간 미리보기 |
| 배경 이미지 | 수동 URL 입력 | 프리셋 선택 + 직접 업로드 |
| 카테고리 | 매번 수동 선택 | 이전 선택 기억, 공지사항 자동 선택 |

---

### 채팅 관리

| 개선 항목 | 이전 | 이후 |
|----------|------|------|
| 화면 구성 | 단일 화면에 모든 기능 | 탭 분리 (채팅방 관리 / 환불 처리) |
| 사용성 | 복잡함 | 목적별 분리로 직관적 |

## 사용 가이드

### iOS 환불 내역 확인하기
1. 관리자 대시보드 접속
2. 좌측 메뉴에서 "iOS 환불 관리" 클릭
3. 필터 (상태, 기간, 검색어) 설정
4. "검색" 버튼 클릭

### 파묘 좋아요 처리하기
1. "[리텐션] 파묘 좋아요" 메뉴 접속
2. 처리할 유저의 "상세" 버튼 클릭
3. 처리할 좋아요 선택 (최대 5개)
4. 거절 비율 조정 (기본 20%)
5. "처리하기" 버튼 클릭

### 배너 등록하기
1. "배너 관리" 메뉴 접속
2. "새 배너 추가" 버튼 클릭
3. 위치 선택 (홈/모먼트)
4. 이미지 업로드
5. 클릭 시 이동 URL 입력 (선택)
6. 노출 기간 설정 (선택)
7. "등록" 버튼 클릭

### 국가 전환하기
1. 화면 우측 하단 국기 아이콘 클릭
2. 원하는 국가 선택 (한국/일본)
3. 자동으로 해당 국가 데이터로 전환

## 메뉴 구조

```
관리자 대시보드
├── 메인 대시보드
├── 회원 통계
├── 커뮤니티 관리
├── 사용자 관리
├── 매칭 관리
├── 신고 관리
├── 채팅 관리
├── AI 채팅 관리
├── 버전 관리
├── SMS 관리
├── 푸시 알림 관리
├── 카드뉴스 관리
├── 배너 관리          ← 신규
├── 매출 조회          ← 결제 성공률 추가
├── iOS 환불 관리      ← 신규
├── 회원 적격 심사     ← 개선
├── 여성 유저 리텐션
├── [리텐션] 탈퇴 회원 복구  ← 신규
├── [리텐션] 파묘 좋아요     ← 신규
├── [리텐션] 처리 이력       ← 신규
└── 구슬 관리
```
